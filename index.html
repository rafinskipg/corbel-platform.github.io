Corbel

Backend that suits for any application.

Corbel platform is a generic platform that offers you all you need to build any application such as authentication, resources or events.

Get started

Made by developers for developers

Corbel platform provides all you need to develop any application quickly and easily.

Open source

Developed under Apache 2 license and built using JVM Languages such as Java or Scala.

Build tools

Corbel platform uses travis, coveralls and can be found packaged in the Sonatype repositories.
